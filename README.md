# dGC-JP: 日本版デジタル成長曲線 (Digital Growth Chart for Japan)

日本人の成長基準に基づく成長指標の算出および可視化プラットフォームです。

## 概要

本ツールは、小児科医や保健指導に携わる専門家が、日本人児童の成長データを迅速に評価・可視化するために開発されました。入力されたデータに基づき、身長・体重のSDS（標準偏差スコア）や肥満度、身長速度（HV）を自動計算し、学術臨床基準に完全に準拠した成長曲線上にプロットします。

## 主な機能

- **成長曲線の描画**: 日本人の最新基準値（2016年）に基づく 0SD, ±2SD, ±2.5SD, ±3.0SD の基準線表示。
- **指標の自動算出と詳細な年齢表示**:
  - **身長SDS**: 日本小児内分泌学会（JSPE）「附表１」の満月齢別平均値・標準偏差（2000年版）に基づき緻密に算出。
  - **体重SDS & 成長曲線背景**: 最新のLMS法による高精度補間算出（2016年版）。
  - **年齢併記表示**: 臨床的に厳密な「小数点4桁表示（例：3.5000歳）」と、直感的かつ簡潔な「満年齢月齢表記（例：満3歳6ヶ月）」を完全に併用。余計な通算月数表示（例：77ヶ月）を排除し、極めて見やすく洗練されたUIです。
  - **肥満度（身長別標準体重・年齢別標準体重）**
  - **身長速度 (Height Velocity: HV) および その SDS**（Suwaら, 1992年の縦断的成長基準に準拠）
- **スマートなデータ入力**: キーボード操作（Enterキーでのフォーカス移動）のみで完結する高速入力インターフェース。
- **修正月齢対応**: 早産児（37週未満）の場合、修正月齢でのプロットおよび評価が可能。修正年齢に対しても、同様に小数点4桁と簡潔な満年齢月齢表記を併記。
- **データのエクスポート/インポート**: 入力したデータをJSON形式で保存・読み込み可能。
- **印刷・画像出力**: 成長曲線の高解像度PNG保存および印刷用レイアウト出力。

## 参考文献・出典

本ツールのロジックは以下の臨床基準に基づいています：

- **身長SDS（満月齢基準）**: [日本小児内分泌学会 附表１（平均体重／標準偏差 2000 年）](https://jspe.umin.jp/medical/files/fuhyo1.pdf)
- **成長曲線・体重SDS**: Clin Pediatr Endocrinol 25:71-76, 2016
- **肥満度計算**: Clin Pediatr Endocrinol 25:77-82, 2016
- **身長速度SDS**: Clin Pediatr Endocrinol 1(1):5-13, 1992 (Suwaらによる縦断的成長データ)

## プライバシーとセキュリティ

- **ローカル完結型**: 入力された患者IDや測定データはすべてブラウザ内蔵のメモリで処理されます。外部サーバーへ送信・蓄積されることはありません。
- **揮発性**: セキュリティのため、ブラウザをリロード（再読み込み）すると入力データは破棄されます。データの保存が必要な場合は、ツール内の「保存」機能を利用してください。

## 開発環境と品質管理

- **フレームワーク**: React 19 (TypeScript)
- **ビルドツール**: Vite
- **可視化**: D3.js (データ可視化)
- **デザイン**: Tailwind CSS, Shadcn UI
- **品質管理**: Vitest による計算ロジック（LMS, 各種SDS, 肥満度、満月齢算出など）の自動テスト実施済み。
